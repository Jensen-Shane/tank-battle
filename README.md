# 🎮 坦克大战 - Tank Battle

一个使用纯 HTML/CSS/JavaScript 制作的经典坦克大战网页游戏。

## 🎯 游戏特色

- 🎨 **精美画面**: Canvas 绘制，包含坦克履带动画、爆炸粒子特效、屏幕震动
- 🎵 **音效系统**: 使用 Web Audio API 实现射击、爆炸、道具等音效
- 💪 **道具系统**: 加速⚡、速射🔥、护盾🛡️、生命❤️ 四种道具
- 🧱 **可破坏地图**: 砖墙可被子弹摧毁，钢墙不可摧毁
- 🤖 **AI 敌人**: 敌方坦克具有追踪玩家和攻击基地的 AI 行为
- 📈 **关卡系统**: 消灭10辆敌方坦克进入下一关，难度递增
- 🏆 **基地保护**: 保护你的基地不被摧毁
- 📱 **移动端支持**: 触屏操作，左半屏移动，右半屏射击
- ⏸️ **暂停功能**: 按 P 键暂停游戏

## 🎮 操作方式

| 操作 | 键盘 | 移动端 |
|------|------|--------|
| 移动 | `WASD` 或 `↑←↓→` | 左半屏滑动 |
| 射击 | `空格` 或 `J` | 右半屏点击 |
| 暂停 | `P` | - |

## 🚀 快速开始

1. 克隆仓库:
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接用浏览器打开 `index.html` 即可开始游戏！

或者直接访问 GitHub Pages: `https://YOUR_USERNAME.github.io/tank-battle/`

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3
- 原生 JavaScript (ES6+)
- Web Audio API

## 📝 游戏规则

1. 控制你的绿色坦克消灭所有敌方坦克
2. 保护底部中央的基地（金色旗帜）不被摧毁
3. 每消灭 10 辆敌方坦克进入下一关
4. 收集道具增强你的战斗力
5. 你只有 3 条命，全部失去则游戏结束
